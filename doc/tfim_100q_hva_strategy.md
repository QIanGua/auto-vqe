# 武器 2：物理先验与平移不变量策略 (HVA)

## 背景：100 量子比特的维度诅咒与贫瘠高原

在处理 100+ 量子比特规模的 VQE 任务时，如果我们依旧采用传统的“全连接随机漫步”策略或基础的 Hardware Efficient Ansatz (HEA)，参数空间会随着系统规模迅速爆炸：

1. **参数泛滥**：即使单层也需要数百个独立的物理参数来控制每个比特。
2. **优化失效**：即所谓的“贫瘠高原” (Barren Plateaus) 现象，当参数达到一定数量和深度时，损失平面的梯度将指数级趋近于零，不论是梯度的正向引导还是随机变异都无法让模型收敛。
3. **内存墙**：基于 PyTorch 的自动微分 (Autograd) 甚至会在计算复杂网络（特别是 100-qubit 的 MPS）反向传播 SVD 时导致严重的运行缓慢甚至内存枯竭。

## 解决途径：Translational Invariance (平移不变量)

为了能在这样的体量下寻获优异的变分线路结构，Agent-VQE 框架实现了基于物理先验的极端限制策略——**平移不变量 (Translational Invariance)**，也称作 Hamiltonian Variational Ansatz (HVA) 的特征提取方式。

我们通过在搜索配置中强行卡死 `param_strategy = "translational"`：
- 意味着这 100 个比特不论横跨多远，**同一层所有的相同操作仅由单一物理参数共享控制**。
- 这不仅将参数规模直接从 $O(N)$ 暴力降维至 $O(1)$，更能保持物理系统的晶格平移对称性。

### 降维打击实例

在一个 3 层的配置下：
```json
{
    "layers": 3,
    "single_qubit_gates": ["ry", "rx"],
    "two_qubit_gate": "rzz",
    "entanglement": "linear",
    "param_strategy": "translational",
    "use_mps": true
}
```
原本这需要数百到一千余个独立参数。但在 `translational` 策略下，由于底层逻辑已被收缩：
- **每一层仅需要 3 个参数**：1 个管所有的 $R_y$，1 个管所有的 $R_x$，1 个管所有的 $R_{zz}$ 纠缠。
- **整个 100 量子比特系统**，只有短短 **9 个参数**。

## 实战表现 (Scipy COBYLA 无梯度收敛)

凭借着仅需 9 个超参的巨大优势，我们**完全跳过了 PyTorch 的反向传播（Backward AD）瓶颈**！
我们在 `core/evaluator/scipy_training.py` 中引入了 `scipy_vqe_train`（运用 COBYLA 等纯前向评估下降的算法），实现了速度与精准度的双重收割：

在无需计算梯度的前提下，该模型仅耗时 **57秒** 就完成了对 100 量子比特矩阵乘积组 (MPS) 的 100 次迭代验证并大幅逼近极小值！

## 适用性与局限性分析

**平移不变量策略并不是“万金油”**，它的核心是对基态结构的强烈物理预判。在自动搜索过程中，Agent 必须根据待求解任务的哈密顿量特性来决定是否使用它。

### 🟢 适用场景 (Highly Applicable)

1. **周期性晶格模型 (Periodic Lattice Models)**：
   如我们目前测试的 Transverse Field Ising Model (TFIM, 横场伊辛模型)、Heisenberg 模型、Hubbard 模型等。这些系统本身就拥有平移对称性（哈密顿量中相邻格点的相互作用强度恒定）。系统的基态自然具有相同的对称性，因此 HVA 能够以 $O(1)$ 的参数极度精准地捕获物理特征。
2. **均质材料与长链体材料 (Homogeneous Materials)**：
   例如无限长的一维原子链或存在明确超胞对称的二维/三维晶格结构。
3. **量子近似优化算法 (QAOA) 处理规则图问题**：
   在解决 MaxCut 等图规划问题时，如果图结构是高度规则和对称的（如强正则图），HVA 绑定的参数依然能迅速找到极值。

### 🔴 不适用场景 (Not Applicable)

1. **无序系统与杂质掺杂 (Disordered Systems / Impurities)**：
   如果在 TFIM 模型中引入了随机的局部磁场干扰（即哈密顿量丧失了平移对称性），或者材料结构中存在点状缺陷，这 9 个参数会将缺陷区域与完好晶格区域同等对待，注定无法精确拟合出真实的基态扰动，此时必须退回层级绑定 (`tied`) 或弱共享模型。
2. **大多数真实分子与量子化学系统 (Quantum Chemistry / Fermion Molecular Models)**：
   像 LiH（氢化锂）、水分子等，虽然分子本身有各种点群对称性，但并不存在一维/二维的平移对称性。核心原子与外围原子的关联相互作用强度天差地别。如果强行对整个分子的 UCC ansatz 中所有的单/双激发项施加同一个旋转角度参数，会直接抹除其独有的化学结构信息，彻底丧失化学精度。
3. **非规则图结构的组合优化**：
   大部分现实中的 VRP (车辆路径问题) 或随机 MaxCut 实例产生的连线全无对称规律可言，强制参数平整化意味着失去了对局部拥堵节点进行特殊调整的能力。

结合 [MPS 模拟器支持] 与这把 [降维武器 `param_strategy: "translational"`]，Agent-VQE 正式跨越了传统算法面对 $N=100$ 时的软硬件双重封锁。但这同时也赋予了架构搜索 Agent 极其重要的分类决策能力：**未来在面对大规模体系时，Agent 必须首先诊断底层哈密顿量的对称群，再决定是掷出 HVA 这个“核武器”，还是老老实实地分配局域自由度。**

## 延展：除了平移对称性，还有哪些极其重要的物理对称性可供剥削？

不仅是“平移对称”(Translational Invariance) 能够实现 $O(1)$ 的究极降维。如果我们能够预先侦测（或由 Agent 智能诊断）出哈密顿量的其他对称性，在设计变分线路 (Ansatz) 及初始态时，就能将其参数空间继续暴削，甚至直接将搜索路线锁定在正确的希尔伯特子空间中。

1. **粒子数守恒 (Particle Number Conservation / $U(1)$ Symmetry)**
   - **典型场景**：量子化学分子（LiH, H2O 等）、Hubbard 模型。电子是不生不灭的。
   - **如何利用**：哈密顿量与总粒子数算符 $\hat{N}$ 绝度对易 ($[\hat{H}, \hat{N}] = 0$)。因此我们只需将初始态准备在正确的 Hartree-Fock (HF) 态上，并**严格限制 Ansatz 只能使用保粒子数的旋转操作**（如 Exchange Gate 或者是 Givens Rotations）。这能直接剔除掉希尔伯特空间中绝大多数完全不合法的状态，极大地压缩搜索截面并预防陷入非物理极小值。

2. **自旋守恒 (Spin Symmetry / 总 $S_z$ 守恒 与 总 $S^2$ 守恒)**
   - **典型场景**：无外加翻转磁场的电子关联体系。往往要求处于单态 (Singlet, $S=0$) 或是三重态 (Triplet, $S=1$)。
   - **如何利用**：如果哈密顿量和总自旋在 Z 轴的投影 $\hat{S}_z$ 以及总自旋平方 $\hat{S}^2$ 对易，那么基态必定处于特定自旋的本征态内。在构造 UCC (Unitary Coupled Cluster) 或硬件高效线路时，可以强行设置 $\alpha$（自旋向上） 和 $\beta$ （自旋向下）轨道的激发参数是对等绑定的。这能把量子化学中所需的非线性优化参数再次**当场减半**。

3. **反映/反演对称性 (Reflection / Inversion Symmetry / Parity $Z_2$)**
   - **典型场景**：诸如一维环、无扰动的小分子，拥有空间镜面对称（左边和右边长得一样）。
   - **如何利用**：像水分子（$C_{2v}$ 点群）或者横场伊辛模型本身就带有奇偶校验的 $Z_2$ 对称。线路中对“晶格左侧”与“晶格右侧”所施加的同类型门，完全可以像使用 `tied` 策略那样共享一组参数。这种拓扑学上的镜像绑定，能够极度弱化长链系统的训练难度。

4. **时间反演对称性 (Time-Reversal Symmetry)**
   - **典型场景**：绝大多数没有外加静磁场的稳定分子或无自旋轨道耦合材料。
   - **如何利用**：在这种假设下，基态波函数的系数必定是**纯实数 (Real-valued)**。因此在设计 Ansatz 时，我们根本不需要引入带有虚数 $i$ 的复杂相位门（如 $P$, $T$, 或者任何会在参数域外产生复数相位的演化），直接全用类似 $R_y$ 的实数算符就可以覆盖全部解空间，将参数量再砍去那些冗余的相位自由度。

**架构搜索启示**：
综上所述，Agent 的任务不应该仅仅是随机拼凑矩阵，它的进阶目标应当是**先用廉价的探测器侦测这 4 大对称性**（计算 $[\hat{H}, \hat{O}_{\text{sym}}]$），然后自动为架构挂载“保粒子数拦截器”、“镜像参数绑定”、“实数编译器”等规则。这些**被物理守恒定律武装过的先验知识**，才是支撑 Agent 攻克数百个量子比特量子化学的终极策略。
